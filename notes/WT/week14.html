GSM Network and Evolution

Review

  • GSM (Global Systems for Mobile Communications) has evolved through the years, progressing from 2G to 4G
  • Key benefits is faster data rates which enables real-time data services on smartphones and mobile equipment
    • Video Streaming
    • MMORPG
    • mobile TV

GSM Network (Week 13)

  • 3 subsystem: MS, BSS, BSS
  • Connects to other networks, such as standard phone network (PTSN), Internet, networks carrying services like video, mobile TV, gaming
  • A mobile station (like a handphone) can be used for:
    • Voice Communication only
    • Data Communication only
    • Both voice and data services

GSM (2G)

  • Operates in:
    • 900MHz and 1800MHz bands in Europe and Asia
    • 850MHz and 1900MHz (or referred as 1.9GHz) bands in USA
  • Security:
    • Uses Encryption to make phone calls more secure
  • Compatibility:
    • Dual-band 850MHz/1900MHz GSM phones used in US are not compatible with the international system
    • For cell phone access outside the US, the following phones
      • tri-band (850/1800/1900MHz) phone
      • quad-band (850/900/1800/1900MHz) phone
  • Data Bandwidth
    • Voice: 14.4kbps
    • Data: 9.6kbps

GPRS (2.5G) over GSM network - Optional

Overlays the existing GSM network to provide high-speed data services

  • Considered a form of data connection using packet radio service over an existing GSM network
  • Bandwidth
    • Approximately up to 48kbps

Purpose:

  • Designed to support bursty application such as emails, broadcast services and web browsing that do not require detected connection

Applications are categorise:

  • Communications: Email
    • Value-added services: Information Services/ Games
    • E-commerce: Ticket purchasing/ Retail/ Banking
    • Location-based services: Navigation; traffic conditions, assisted GPS
    • Vertical Applications: Freight delivery
    • Advertising

GPRS Functional Components

Packets Control Units (PCU)
  • Module within BSC that is responsible for handling packet traffic
Signalling System No. 7 (SS7)
  • A set of telephony signalling protocols which are used to set up most of the worldâ€™s PSTN telephone
Serving GPRS Support Node (SGSN)
  • Responsible for the delivery of packet from and to the mobile stations within its geographical area
  • It tasks include packet routing and transfer, mobility management, logical link management, and authentication and charging function
  • Location Register of SSGN stores location information and users profiles used in the packet data of all GPRS users register within this SGSN
Gateway GPRS Support Node (GGSN)
  • Responsible for interworking between GPRS network and external packet switched networks such as the Internet

3rd Generation Partnership Project (3GPP)

A collaboration between groups of telecommunications association, known as the Organisational Partners

  • Initial Scope:
    • Make a globally applicable third-generation (3G) mobile phone system specifications based on the GSM specification
  • 3GPP standards are structured as releases

UMTS (3G)

Universal Mobile Telecommunication System

  • Extends GSM/GPRS (2.5G) networks and uses W-CDMA air-interface
  • 3G mobile communication system provides a range of low-cost broadband services to the world of wireless and mobile communication
  • Designed to deliver media (pictures, video communication), as well as voice and data, to mobile wireless subscribers
  • Data Rates:
    • 144kbps - Satellites and rural outdoor
    • 384kbps - Urban Outdoor
    • 2048kbps (2Mbps) - Indoor and low range outdoor
  • Bandwidth for digital voice
    • up to 64 kbps
  • Frequency bands are named according to Roman Numeric Number
    • Europe and Asia
      • Band I (W-CDMA 2100MHz)
      • Band VIII (W-CDMA 900MHz)
  • Integration of Voice & Traffic Data
    • Multimedia applications require both voice and data
    • Main feature of 3G cell phone system is to handle both voice and data
    • Data oriented system, e.g 802.11 need to handle voice due to the use of Voice over IP (VoIP)

Some UMTS Function Components

UMTS Terminal

aka User Equipment (UE)

  • A new name is given to what was previously termed mobile station (MS) in GSM network, to reflect the greater functionality of the UE
UMTS RNS (Radio Network Subsystem)

Equivalent to the previous Base Station Subsystem (BSS) in GSM. Provides and manages air interface for overall network

  • Consists of:
    • UMTS BS (Base Station)
      • Also referred to Node B in the context of UMTS
    • UMTS RNC (Radio Network Controller)
      • Undertakes the radio resource management and some of the mobility management functions
      • Point where encryption/decryption is performed to prevent eavesdropping
      • All UMTS radio-resource functions are located in RNC including handoff
      • In GSM, control of handoff process is shared between MSC and BSC

HSPA (3.5G)

High-Speed Packet Access

HSDPA (High-Speed Downlink Packet Access)

  • 3GPP Release 5 introduces HSDPA that allows UMTS-networks to provide higher downlink data transfer rates on smartphones
  • May deliver up to 14.4Mbps download speed
Key Benefits
  • High data transfer rates
  • Lower Latency
    • Useful for time-sensitive applications such as interactive and streaming multimedia
    • Application such as database transactions
  • Better modulation techniques
  • Retransmission from Base Station
  • Scheduling done at Base Staton

HSUPA (High-Speed Uplink Packet Access)

aka Enhanced Uplink (EUL)

  • Max uplink transfer rate is upgraded from 2Mbps to 5.76Mbps with Release 6 in HSUPA for smartphones like the iPhone 4
  • HSDPA and HSUPA are collectively called HSPA

HSPA+ (Evolved HSPA)

Wireless broadband standard defined in 3GPP Release 7

  • Uses MIMO technologies and higher-order modulation (64-QAM)
  • Bandwidth:
    • Downlink - up to 56Mbps
    • Uplink - up to 22Mbos
    • Future bandwidth of 168Mbps may be possible using multiple carriers

LTE (4G)

Long Term Evolution

  • Bandwidth:
    • Downlink - up to 150Mbps

How it works:

Air Interfaces
  • 2 different types of air interface (radio links)
    • One for Downlink
      • One for Uplink
    • Cell tower uses multiplexing technique that requires a power transmitter
    • LTE devices do not have a strong and powerful signal going back to the cell tower, to conserve battery life
    • LTE devices use a different multiplexing scheme that does not require a powerful transmitter
Multiplexing Technique
  • LTE uses OFDMA (orthogonal frequency division multiple access)
    • The OFDM signal used in LTE comprises a maximum of 2048 different sub-carriers with a spacing of 15kHz
    • Within the OFDM signal, it possible to choose 3 different types of modulation, depending on the quality of the signal
      • QPSK
      • 16-QAM
      • 64-QAM
Multiple-input and Multiple-output Antenna (MIMO)
  • MIMO devices have multiple input and output antennas
    • These devices have multiple connection to a single cell
      • Increase stability of the connection
      • Reduces latency tremendously
    • Increase total throughput of a connection

LTE+ (4G+)

LTE-Advanced

  • Bandwidth:
    • Downlink - up to 300Mbps

Technology Used

MIMO

Combines multiple antenna on both the transmitter (4G tower) and the receiver (smartphone)

  • So a 2x2:2 MIMO means there are 2 antennas on the transmitter and 2 antennas on the receiver, transmitting and receiving 2 independent data streams
  • Therefore, more antenna means faster potential speeds as data streams can travel more efficiently
Carrier Aggregation

Allows devices to receive multiple different 4G signals at once

  • Signals do not have to be on same frequency, meaning one can receive an 1800MHz and 800MHz signal at the same time, which is not possible with standard 4G
  • Up to 5 different signals can be combined at once with each offering 20MHz bandwidth, where they can be combined to create a data pipe of up to 100MHz of bandwidth
Relay Node

Low power base stations that provide enhanced coverage and capacity at cell edge, and hot-spot areas and also be used to connect to remote areas without fibre connection

  • In networks with a mix of large and small cell, performance can be improve via introduction of Relay Nodes (RN)

5G

Aims for:

  • Higher capacity than current 4G
    • Higher number of mobile broadband users per are unit
    • Allow consumption of higher or unlimited data quantities in gigabyte per month
    • Feasible for large portion of population to stream HD media many hours a day, when out of reach of WiFi hotspots
  • Improve support of machine-to-machine (M2M) communication (aka Internet of Things)
  • Lower cost
  • Lower battery consumption
  • Lower latency than 4G equipment

Standards

  • No current standards for 5G deployments
  • Next Generation Mobile Network Alliance defines that a 5G standard should fulfil:
    • Data rates of 100Mbps for metropolitan areas
    • 1Gbps simultaneously to many workers on the same office floors
    • Several hundred of thousands of simultaneous connections for massive wireless sensor network
    • Spectral efficiency significantly enhanced compared to 4G
    • Improve Coverage
    • Signalling efficiency enhanced
    • Latency reduced significantly compered to LTE
