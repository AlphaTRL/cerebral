Cellular Network Technology I

tldr - The cellular concept, the architecture of the GSM networks, the GSM network layer, and GSM radio link/interface

Cell Phones

Communication

Cellphone network uses cell towers (antenna) via a switching office to communicate via radio waves

Transmission

  • For transmission using radio waves, a frequency channel is used
  • As a cell phone is a duplex device, it can transmit and receive at the same time. Hence 2 channels are required for each phone
  • Disadvantage of having 1 cell tower in a whole city
    • Not enough channels to support many users
    • Each cell phone must have a powerful transmitter and receiver in order to reach the cell tower (more power consumption)

Cell Concept

To overcome the limitation of 1 large central antenna, cell concept is used

  • A geographical area is divided into small areas called cells
  • Cells can be sized at about 2 x 2 square miles
  • Cells are shown as hexagons on a hexagonal grid

Low Power Transmitter

Each cell has a base station that consists of an antenna tower and a small building containing the radio equipment. The base station uses a low power transmitter

  • Low power transmitter has 2 advantages:
    1. Battery Life
      • Reduces power consumption of a cell phone, which is normally battery operated. Low power means small batteries.
    2. Frequency Re-use
      • Transmissions between a base station and phones with its cell do not make it very far outside the cell. Non-adjacent cells can use the same set of frequencies without interference, and can be reuse across a big area. This reduces the number of frequencies to support all users.

Base Station

A base station manages, sends, and receives traffic from mobile phones within the geographical area to a distant cellular telephone switch called a Mobile Telecommunications Switching Office (MTSO)

  • MTSO
    • Authenticates wireless customers before establishing connection
    • Places calls from land-based telephones to wireless customers
    • Switches calls between cells as mobile travel across cell boundaries

Cellular Phone Technologies

Widely used cellular technologies are:

CDMA (Code Division Multiple Access)

TDMA (Time Division Multiple Access)

Analog and inefficient. Also obsolete

GSM (Global System for Mobile Communications)

Adopted by Singapore

GSM Networks

GSM Network Architecture

Mobile Station

  1. Mobile Equipment (ME)
    • Identified internally by IMEI (International Mobile Equipment Identity), which is allocated by equipment manufacturer
    • IMEI is registered into Equipment Identity Register (EIR) by the network operator (telco)
    • Example: Handphones
  2. Subscriber Identity Module (SIM)
    • Contains IMSI (International Mobile Subscriber Identity) that identifies a user on a network.
    • IMSI contains Mobile Country Code (MCC), Mobile Network Code (MNC) and Mobile Subscriber Identity Number (MSIN)

Base Station Subsystem

  1. Base Transceiver Station (BTS) or Base Station
    • Helps define a cell’s boundary, and responsible for the radio link protocol with the Mobile Station
    • Transmit and receives signal to/from mobile users using different frequencies
    • Found on elevated location such as towers, poles, water tanks or roof-tops
    • Has a frequency band that provides multiple channels
  2. Base Station Controller (BSC)
    • Controls multiple BTSs
    • Manages radio channel setups and hangovers
    • BSC connects the Mobile Station and the Mobile Switching Centre (MSC)

Network Switching Subsystem

  1. Mobile Switching Centre

    Also known as Mobile Telephone Switching Office (MTSO)

    • Handles a few BSCs
    • Manages location of mobiles
    • Manages setup of calls to/from mobile station
    • Manages security
    • Collects call billing data and sends to billing system
    • Collects traffic statistics for performance monitoring
  2. Home Location Register (HLR)

    Stores location about each subscriber that belongs to the MSC

  3. Visitor Location Register (VLR)

    Maintains information about subscribers currently physically in the region

  4. Authentication Centre (AuC)

    Databases used for handling authentication and encryption keys

  5. Equipment Identity Register (EIR)

    A list of all valid mobile equipments (ME) on the network by keeping all valid IMEI in used

GSM Signalling Protocols

GSM components talk to each other using signal-based protocols

  • Signalling Protocols involve 3 layers
    • Layer 1: Physical Layer
    • Layer 2: Data Link Layer
    • Layer 3: Network Layer

The Network Layer (Layer 3) is divided into 3 functionalities:

Radio Resource Management (RRM)

Manage call control, supplementary service and short message services (SMS). E.g. establishes incoming call connection

  • Involves 3 functional components
    1. Mobile Station (MS)
    2. Base Station Subsystem (BSS)
    3. Mobile Switching Centre (MSC)
  • A major aspect of RRM is handover or handoff

    The switching of an ongoing call to a different channel or cell

Types of handover:

Internal handover (handled by BSC)

Intra-cell handover

Transferring call connection to another channel on the same BTS

Intern inter-cell handover

Transferring call connection to another BTS on the same BSC

External handover (handled by MSC)

MSC intern handover

Transferring call connection between BTSs of different BSCs on the same MSC

MSC extern handover

Transferring call connection to another BTS under the control of different MSC

Mobility Management (MM)

  • Handles the following issues
    • Location Management and other mobility issues
    • Authentication and other security issues
  • Main function is to facilitate location updating

    When there is an incoming call for a mobile station (MS1), the network needs to be aware of the location of MS1 so as to transfer the call to the proper cell

  • Location updating involve 3 components
    1. Home Location Register (HLR)
    2. Visitor Location Register (VLR)
    3. Mobile Switching Centre (MSC)
How MM works:
  1. In GSM, cells are group into location areas (LA)
  2. When a Mobile Station (MS) crosses from one LA to another, it sends location updating messages to the MSC and VLR of the new LA
  3. The new VR records the location information of the MS
  4. Location Information is sent to the HLR from the new VLR
  5. The HLR sends a subset of the subscriber’s information to the new VLR required for call control
  6. The HLR informs the previous MSC/VLR to cancel registration of the MS

Connection Management (CM) - Optional

Overseas link between mobile and MSC. E.g. manages handovers

Radio Link/ Air Interface

  • 2 Frequency Bands are used:
    1. Uplink Transmission (MS to BS)
    2. Downlink Transmission (BS to MS)
  • In GSM-900, a frequency band of 25MHz are allocated to each links
    1. Uplink (890 - 915 MHz)
    2. Downlink (935 - 960 MHz)
  • GSM uses a combination of FDMA and TDMA
  • Each 25MHz bands are divided into 124 frequency sub-bands (124 channels) using FDMA
  • Each channels is allocated a 200kHz carrier spacing
  • The 2 frequency bands give a total of 248 channels (2 * 124)
  • 248 channels, combined with an 8-slot TDMA scheme, gives a total of 1984 logical channels
  • If there is a cluster of 7 cells sharing the logical channels, each cells in the cluster would use 283 channels
